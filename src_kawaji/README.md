## For activate and use virtual environment
Step 1
```
sudo apt install python3.12-venv
```
```
python3 -m venv venv
```

Step 2
```
source venv/bin/activate
```

Step 3
```
pip install -r requirements.txt
```

## pre-commit init
```
pre-commit install
```

## pre-commit check
```
pre-commit run --all-files
```
